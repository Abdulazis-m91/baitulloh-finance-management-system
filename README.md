# 📌 Overview

Baitulloh Finance Management System is a web application built to simplify financial management for foundations.  
This system helps administrators or treasurers record transactions, monitor financial activities, and manage financial reports in a structured and efficient way.

The project demonstrates how a digital system can improve **financial transparency, data organization, and reporting efficiency**.

---

# ✨ Key Features

### 📊 Financial Dashboard
Displays financial summaries and important financial information.

### 💰 Income Management
Manage and record all incoming funds.

### 💸 Expense Management
Track operational expenses and financial transactions.

### 📑 Financial Reports
Generate and review structured financial reports.

### 🗂 Transaction History
Store and review historical financial records.

### 👤 Admin Dashboard
Centralized control panel for managing financial data.

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/Abdulazis-m91/baitulloh-finance-management-system.git
```

Go to the project directory

```bash
cd baitulloh-finance-management-system
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
